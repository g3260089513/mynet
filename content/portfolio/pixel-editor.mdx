## 项目概述

像素编辑器是一款纯前端工具，利用 HTML5 Canvas API 实现完整的像素画创作体验。支持图层管理、自定义调色板、动画帧编辑和多种导出格式。

### 技术亮点

- **Canvas 优化**: 使用离屏 Canvas 实现图层合成，保证大画布下的流畅操作
- **状态管理**: Zustand 管理复杂的编辑状态，包括图层、帧和历史记录
- **性能优化**: 通过脏矩形检测，只重绘发生变化的区域
