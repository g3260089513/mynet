## 项目概述

游戏数据分析仪表盘是一个面向核心玩家的数据追踪平台。它聚合了 Steam、PlayStation 和 Xbox 的游戏数据，提供统一的游戏时间追踪、成就进度监控和战绩分析。

### 技术亮点

- **实时数据同步**: 使用 Supabase 的实时订阅功能，数据变化即时反映在仪表盘上
- **可视化图表**: D3.js 构建自定义图表，展示游戏趋势和统计分布
- **响应式设计**: 从手机到宽屏，完整的响应式体验

### 挑战与收获

最大的挑战是处理不同平台的 API 差异和数据格式。通过抽象出统一的数据层，我成功地将三个完全不同平台的数据整合到同一个视图。这个项目让我深入理解了实时数据处理和可视化设计。
